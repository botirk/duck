# Функции
## Примеры реализации
```
pow = (a) => {
  < a * a
}
```
Как  вы видите, нет ключего слова **return**, вместо него используется символ **<**
<br><br>
```
pow = a => a * a
```
Сокращённо
<br><br>
```
pow = a => a * a
pow("Hello World!")
```
Ошибка во время компиляции в отличии от runtime exception lua/js
<br><br>
```
pow = (a: number) => a * a
```
Так тоже можно
